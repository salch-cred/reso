Compliance that proves itself,

Reso lets stablecoin and token issuers on Stellar verify every transaction is KYC'd, sanctions-clear, and within limits — using zero-knowledge proofs, not a public ledger of who owns what.

Designed for the SEP-0057 compliance-hook standard, built on Stellar's native zero-knowledge primitives (Protocol 25)
Sanctions Screening

Screen every transfer, reveal nobody's identity

Reso checks each transaction against a live sanctions list using a zero-knowledge non-membership proof — the chain learns "clear" or "blocked," never who was checked.

Spending Limits

Limits enforced without ever seeing a balance

Daily and per-transaction caps run against an encrypted running total, so not even Reso or the issuer can see a user's spending pattern — only the pass/fail result reaches the chain.
